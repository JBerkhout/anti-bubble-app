Class ClassOverviewComponent

This class handles all the logic for the overview of a student's class where all the student's classmates are shown. The class contains helper methods used in the HTML of the componenet to filter thorugh classmates and also a method to open a MatDialog to leave a class, defined as a sub-component in the file.
