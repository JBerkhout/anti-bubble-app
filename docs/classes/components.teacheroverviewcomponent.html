Class TeacherOverviewComponent

This class handles all the logic for the class-overview for a teacher. The class contains methods for creating a class, switching between teacher owned classes and displaying all the class members of the selected class. Class contains two helper methods for opening MatDialogs for kicking students out of a class and deleting a class, defined as sub-components in the file.
